## make
### Linux
```
mkdir build
cd build
cmake ..
make
```
## run
```
./bin/PLANNER
```
ps: please run the programme at the folder `"motion-planning/build"` of run it at `"motion-planning/build/bin"` with `./PLANNER`      

input `ctrl+c` to exit
